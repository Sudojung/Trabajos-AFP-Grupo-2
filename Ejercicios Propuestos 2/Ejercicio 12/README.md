12. Una compañía desea transmitir datos por teléfono pero están preocupados de que sus teléfonos estén "pinchados". Todos sus datos es transmiten como enteros de cuatro dígitos. Se le ha pedido a usted escribir un programa que cifre los datos para poderlos transmitir con mayor seguridad. Su programa deberá leer un entero de 4 dígitos y cifrarlo como sigue: sustituya cada dígito por (ese_digito+7)%10. Luego intercambie el primer y tercer dígitos luego el segundo y cuarto dígitos. Luego imprima el entero cifrado. Escriba otro programa que recoja un entero de cuatro dígitos cifrado y lo descifre para formar el número original.
